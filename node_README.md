# Node Operation: Sovereign-Talent-Engine
# Location: Distributed Mac OS X Stations

## I. Human-in-the-Loop Operations
- **node_discovery.py**: Primary Operator interface. Handles 'Glance-and-Tap' URL verification and fulfills the Hybrid Rotational Survey.

## II. Automated Maturation
- **node_maturated.py**: Background daemon. Performs session-based scraping and deep-text analysis on verified URLs to update the 20-point model.

## III. Local Analytics & Feedback
- **node_report_stats.py**: Tracking for operator performance: verification speed, accuracy ratings, and total substrate contribution.
